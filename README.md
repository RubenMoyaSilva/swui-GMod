# SWUI Content Pack

Paquete de contenido para **Garry's Mod** diseñado para servidores de **Star Wars Roleplay**.

Este repositorio reúne todos los recursos utilizados por el ecosistema **SWUI**, incluyendo modelos, materiales, texturas, sonidos, partículas e imágenes de la interfaz.

> **Este addon únicamente contiene recursos (content).** No incluye lógica de juego ni scripts.

---

# Características

* Recursos de alta calidad inspirados en Star Wars.
* Modelos personalizados.
* Materiales y texturas.
* Sonidos personalizados.
* Efectos de partículas.
* Recursos para HUD e interfaz.
* Iconos para armas y SWUI.
* Optimizado para servidores dedicados de Garry's Mod.

---

# Contenido

Este paquete incluye diferentes tipos de recursos, entre ellos:

* Modelos (`models`)
* Materiales (`materials`)
* Texturas (`VTF` / `VMT`)
* Sonidos (`sound`)
* Partículas (`particles`)
* Recursos para la interfaz de SWUI
* Iconos
* Miras y texturas para armas

# Instalación

## Servidor

1. Descarga o clona este repositorio.
2. Copia la carpeta del addon dentro de:

```text
garrysmod/addons/
```

3. Reinicia el servidor.

## Workshop

Si el addon está publicado en Steam Workshop, simplemente suscríbete a él y asegúrate de que el servidor lo descargue correctamente.

This repository contains all required assets used by the SWUI ecosystem, including models, materials, particles, sounds and interface resources.

* Particle effects
